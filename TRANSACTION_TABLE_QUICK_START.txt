## ↑↓ SORTING EXAMPLES

┌────────────────────────────────────────────────────────────────────────┐
│ Click column headers to change sort order                              │
├────────────────────────────────────────────────────────────────────────┤
│                                                                        │
│ Column   | First Click    | Second Click   | Third Click            │
│ ─────────────────────────────────────────────────────────────────────  │
│ Date     | Newest First ↓ | Oldest First ↑ | None (⇅)               │
│ Merchant | A to Z (A-Z ↑) | Z to A (Z-A ↓) | None (⇅)               │
│ Amount   | Low to High ↑  | High to Low ↓  | None (⇅)               │
│ Source   | A to Z (A-Z ↑) | Z to A (Z-A ↓) | None (⇅)               │
│                                                                        │
│ Visual Indicators:                                                     │
│   ↑  = Ascending sort (A-Z, low to high, oldest first)               │
│   ↓  = Descending sort (Z-A, high to low, newest first)              │
│   ⇅  = No sort applied                                               │
│                                                                        │
└────────────────────────────────────────────────────────────────────────┘

## 🛠️ TROUBLESHOOTING

┌────────────────────────────────────────────────────────────────────────┐
│ Problem: "Failed to load transactions"                                 │
├────────────────────────────────────────────────────────────────────────┤
│ Solution:                                                              │
│   1. Make sure backend is running (python app.py)                     │
│   2. Check it says "Running on http://127.0.0.1:5000"                │
│   3. Click "🔄 Refresh" button in transaction table                   │
│   4. Check browser console (F12) for errors                           │
│                                                                        │
├────────────────────────────────────────────────────────────────────────┤
│ Problem: "No transactions found"                                       │
├────────────────────────────────────────────────────────────────────────┤
│ Solution:                                                              │
│   1. Ingest PDFs first: python ingest_transactions.py                 │
│   2. Wait for ingestion to complete                                   │
│   3. Click "🔄 Refresh" in transaction table                          │
│   4. Transactions should appear                                       │
│                                                                        │
├────────────────────────────────────────────────────────────────────────┤
│ Problem: Search not working                                            │
├────────────────────────────────────────────────────────────────────────┤
│ Solution:                                                              │
│   1. Check typing is correct (case-insensitive)                       │
│   2. Click "Clear" button to reset                                    │
│   3. Try partial matches: "Starbucks" → "Starb"                      │
│   4. Check browser console for JavaScript errors                      │
│                                                                        │
├────────────────────────────────────────────────────────────────────────┤
│ Problem: Sorting not working                                           │
├────────────────────────────────────────────────────────────────────────┤
│ Solution:                                                              │
│   1. Click column header again (toggle sort)                          │
│   2. Third click removes sort (shows ⇅ indicator)                     │
│   3. For amounts: numeric sort (450 > 100)                           │
│   4. For text: alphabetic sort (A < Z)                               │
│                                                                        │
└────────────────────────────────────────────────────────────────────────┘
